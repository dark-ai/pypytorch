### Installation

---

#### PyPI

1. `pip3 install pypytorch`

#### From Source

1. `git clone https://github.com/dark-ai/pypytorch.git`
2. `cd pypytorch`
3. `pip install -r requirements.txt`
4. `make install && make clean`
